Detailed Description

A struct encapsulating an Objects UUID and a pointer to an associated ColoredText.

This struct is for use with ActiveGuiElement instances which present some form of list of objects. It provides a convenient way of linking (for example) game entities to thier associated list entry by associating their UUID with a ColoredText.
