Detailed Description

This class provides the render() and update() methods for the game loop, as well as holding the loaded Actors, Map and providing functions for GUI handling.

Member Function Documentation

void Engine::render ( )

Rendering function.

This function performs the rendering. Actors and the map are rendered on gameConsole, UI is rendered on uiConsole, and then both are blitted onto the root console.
