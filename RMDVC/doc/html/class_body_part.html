Detailed Description

The code representation of a part of the body (such as 'Left Arm').

This class represents the upper level of the body definition. Every BodyPart connects upstream to one BodyPart (except for the root) and downstream to either one or more BodyPart objects or one or more Organ objects. It cannot hold both BodyPart and Organ children, because Organs represent the 'leaves' of the body tree. (No branches grow from leaves, right?)

Instances of this class contain a TCODList of Part objects, which are the Organs or BodyParts connected to this one.
