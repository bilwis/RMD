Detailed Description

The 'leaves' of the Body tree.

This class represents the second-deepest level of the body definition, "above" the Tissue objects and "below" the BodyPart objects. This definition of 'Organ' is very much different from the common meaning of 'Organ', as in internal organs such as the stomach or liver. Instances of this class contain an array of tissue definitions, a reference to the organ to which this one is connected (upstream root) and a list of references to the organs which are connected to this one (downstream branches).
